# Technical Infrastructure Playbook

## Objective

Keep YieldWeek's technical stack reliable, simple, and ready for city execution.

## Core gates

- singapore.yieldweek.org live before launch
- Stripe and Airwallex flows tested before payment collection
- Luma configuration reviewed before registration opens
- attendee data architecture documented before tickets go live

## Stack areas

- websites and landing pages
- Luma event setup
- Stripe / Airwallex payments
- attendee data exports and ownership
- CRM / Google Sheets / HubSpot integrations
- PWA roadmap

## Weekly checklist

1. Review infrastructure log.
2. Check blocking technical dependencies.
3. Confirm payment flow status.
4. Confirm Luma readiness.
5. Review data/privacy issues with Legal.
6. Maintain fallback plans.

## Decision standard

Reliability over features. Own the data. Defer complexity.
