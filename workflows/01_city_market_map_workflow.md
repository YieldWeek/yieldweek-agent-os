# City Market Map Workflow

## Purpose

Build a source-backed view of a target city edition opportunity.

## Trigger

A city is selected for YieldWeek validation or launch planning.

## Agents involved

- Primary agent: Market Intelligence Agent
- Supporting agents: Editorial Agent, Reality Checker Agent
- Reviewer: Orchestrator Agent

## Inputs

- Target city
- Target date window
- Priority audience segments
- Known anchor events
- Current validation goal

## Steps

1. Define research scope
   - Owner: Orchestrator Agent
   - Output: city, date window, target categories, required depth

2. Map events and organisers
   - Owner: Market Intelligence Agent
   - Output: event target list with source links

3. Map sponsors and media partners
   - Owner: Market Intelligence Agent
   - Output: sponsor and media target list

4. Identify narrative themes
   - Owner: Editorial Agent
   - Output: city-specific editorial angles

5. Verify claims
   - Owner: Reality Checker Agent
   - Output: source-backed version with uncertainty labels

6. Recommend next actions
   - Owner: Orchestrator Agent
   - Output: top 10 targets and immediate outreach plan

## Human approval points

Founder approval required before outreach to Tier A targets or public claims about the city opportunity.

## Outputs

- City market map
- Event target list
- Sponsor target list
- Editorial angle list
- Top 10 next actions

## Success metrics

- qualified targets identified
- verified contact paths
- clear calendar white space
- actionable first outbound list
