# Franchisee Qualification Workflow

## Purpose

Monitor, qualify, and eventually onboard franchisee opportunities without activating the franchise model before the edition playbook is ready.

## Trigger

A franchisee prospect is identified or inbound franchise interest is received.

## Agents involved

- Primary agent: Franchisee Manager Agent
- Supporting agents: Product Agent, Legal / Compliance Agent, Commercial Agent, Reality Checker Agent, Orchestrator Agent

## Inputs

- Prospect name
- City
- Operator profile
- Existing event/business interests
- Contact path
- Playbook readiness status
- Territorial conflict risk

## Steps

1. Check playbook readiness
   - Owner: Product Agent
   - Output: ready / not ready for franchise discussion

2. Qualify prospect
   - Owner: Franchisee Manager Agent
   - Output: franchisee fit memo

3. Check legal and territorial risk
   - Owner: Legal / Compliance Agent
   - Output: legal/territorial risk notes

4. Check commercial conflict
   - Owner: Commercial Agent
   - Output: sponsor/direct-operate conflict notes

5. Reality and brand check
   - Owner: Reality Checker Agent
   - Output: claim and brand review

6. Founder decision
   - Owner: Orchestrator Agent
   - Output: hold, monitor, engage, or reject

## Human approval points

Founder approval required before any franchise conversation, fee discussion, territorial discussion, brand usage approval, or franchise agreement is shared.

## Outputs

- Franchisee fit memo
- Playbook readiness status
- Legal/commercial risk notes
- Decision status
- Next action

## Success metrics

- no premature franchise promises
- only qualified operators advance
- playbook readiness gates respected
- brand quality protected
