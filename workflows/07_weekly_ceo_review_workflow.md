# Weekly CEO Review Workflow

## Purpose

Create a weekly operating review that keeps YieldWeek focused on validation, revenue, execution, and risk.

## Trigger

Weekly review cycle or founder request.

## Agents involved

- Primary agent: Orchestrator Agent
- Supporting agents: All agents

## Inputs

- Event pipeline
- Sponsor pipeline
- Organiser pipeline
- Speaker database
- Current priorities
- Open risks
- Previous week commitments

## Steps

1. Collect updates
   - Owner: Orchestrator Agent
   - Output: request for agent status updates

2. Pipeline review
   - Owner: Commercial Agent
   - Output: badge and sponsor pipeline status

3. Market update
   - Owner: Market Intelligence Agent
   - Output: new targets, changes, and market signals

4. Editorial update
   - Owner: Editorial Agent
   - Output: content, speaker, and narrative status

5. Operations update
   - Owner: Operations Agent
   - Output: execution status and blockers

6. Risk review
   - Owner: Reality Checker Agent
   - Output: factual, brand, legal, and commercial risks

7. CEO synthesis
   - Owner: Orchestrator Agent
   - Output: weekly priorities, blockers, decisions needed, next actions

## Human approval points

Founder approval required for priority changes, spend, external commitments, pricing, and strategic pivots.

## Outputs

- Weekly CEO review memo
- Top 5 next actions
- Decision list
- Risk register update
- Pipeline summary

## Success metrics

- clear weekly priorities
- fewer stale tasks
- fast escalation of blockers
- validation gates visibly progressing
